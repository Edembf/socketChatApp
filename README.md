# SocketChatApp

## Overview
SocketChatApp is a JavaScript-based real-time chat application built using Socket.IO, allowing users to communicate in real time with ease and efficiency.

## Features
- Real-time messaging
- User authentication and management
- Join/leave chat rooms
- Message history
- Responsive design for mobile and desktop

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Edembf/socketChatApp.git
   cd socketChatApp
   ```

2. **Install dependencies**:
   Ensure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Run the application**:
   ```bash
   npm start
   ```
   Navigate to `http://localhost:3000` in your browser to access the chat application.

## Usage Information
- After starting the application, users can create their accounts or log in to existing ones.
- Once logged in, users can join specific chat rooms to start sending and receiving messages in real-time.
- The application provides a simple and intuitive interface for seamless communication.
